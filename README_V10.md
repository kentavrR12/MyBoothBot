# Exhibition Booth Configurator Bot v10 (FINAL)

## ✅ Что исправлено в v10

### 🔴 Проблема в v1-v9:
- Данные терялись между модулями (states.py → calculator.py → keyboards.py)
- Расчет стоимости не срабатывал, потому что функция не получала корректные данные
- Ошибки были скрыты в логах, что затрудняло отладку

### 🟢 Решение в v10:
- **Монолитный файл** `main.py` - вся логика в одном месте
- **Гарантированное хранение данных** - все параметры (длина, ширина, тип, материалы, оборудование) хранятся в FSM контексте и **никогда не теряются**
- **Прямой расчет** - функция `calculate_cost()` имеет прямой доступ ко всем данным
- **Логирование в Telegram** - если что-то пойдет не так, бот напишет об этом в чат
- **Полная мультиязычность** - EN, RUS, LV с ценами на кнопках

## 📱 Как использовать бот

1. **Отправить `/start`** - выбрать язык (EN/RUS/LV)
2. **Выбрать длину** - 2-6 метров
3. **Выбрать ширину** - 2-5 метров
4. **Выбрать тип конструкции** - Standard (500€) или Exclusive (800€)
5. **Выбрать материалы** - можно несколько: Plastic (50€), Wood (150€), Metal (200€)
6. **Выбрать оборудование** - можно несколько: Lighting (300€), Furniture (400€), Monitor (800€)
7. **Нажать "Calculate Total"** - получить смету стоимости
8. **Подтвердить заказ** - получить финальный отчет

## 💰 Формула расчета стоимости

```
TOTAL = (Base Price × Area) + (Materials × Area) + Equipment

Где:
- Base Price = 500€ (Standard) или 800€ (Exclusive)
- Area = Length × Width
- Materials = сумма цен выбранных материалов × площадь
- Equipment = сумма цен выбранного оборудования (без умножения на площадь)
```

### Пример:
```
Длина: 3м
Ширина: 4м
Площадь: 12м²
Тип: Standard (500€)
Материалы: Wood (150€)
Оборудование: Lighting (300€)

Расчет:
- Base: 500€ × 12м² = 6000€
- Materials: 150€ × 12м² = 1800€
- Equipment: 300€
- TOTAL: 6000€ + 1800€ + 300€ = 8100€
```

## 🔍 Отладка

### Логи в терминале

Все действия логируются:
```
✅ COST CALCULATED: Base=6000.00€, Materials=1800.00€, Equipment=300.00€, TOTAL=8100.00€
🔍 EQUIPMENT DONE - Data: L=3.0, W=4.0, Type=standard, Mat=['wood'], Eq=['lighting']
```

### Если расчет не работает

1. Проверьте, что бот получил все параметры:
   - Длина (length)
   - Ширина (width)
   - Тип конструкции (construction_type)
   - Материалы (materials) - может быть пустой список []
   - Оборудование (equipment) - может быть пустой список []

2. Проверьте логи в терминале - там будет точная информация об ошибке

3. Убедитесь, что BOT_TOKEN правильный в `.env`
